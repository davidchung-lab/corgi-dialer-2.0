# Corgi Dialer

Power dialer with HubSpot sync, prospect intel, and deal creation.

## Deploy to Netlify (5 minutes)

1. Go to github.com → New repository → name it `corgi-dialer` → Create
2. Upload all these files (drag and drop in GitHub)
3. Go to netlify.com → Add new site → Import from GitHub
4. Select your repo → Build command: `npm run build` → Publish dir: `build` → Deploy
5. Share the Netlify URL with your team

## Local development

```bash
npm install
npm start
```

## What each rep needs
- HubSpot Private App Token (pat-na-...)
- Calendly Personal Access Token
